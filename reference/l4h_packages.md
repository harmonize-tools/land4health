# List all *land4health* packages

List all *land4health* packages

## Usage

``` r
l4h_packages(include_self = TRUE)
```

## Arguments

- include_self:

  default `TRUE`. Includes the "land4health" package name in the
  resultant character vector.

## Value

A character vector of package names included in the "land4health"
meta-package.

## Examples

``` r
l4h_packages()
#> [1] "rgee"        "sf"          "land4health"
```
